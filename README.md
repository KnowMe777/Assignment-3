# Assignment-3

A personal portfolio website built with HTML and TailwindCSS.

## Description

This is a responsive portfolio website showcasing information about Hanan Khan, including sections for About Me, Skills, Education, and Contact information.

## Features

- Responsive design using TailwindCSS
- Mobile-friendly navigation with hamburger menu
- Multiple pages for different sections
- Clean and modern UI

## Structure

```
├── index.html          # Main landing page (About Me)
├── pages/
│   ├── skills.html     # Skills section
│   ├── education.html  # Education section
│   └── contact.html    # Contact information
└── images/             # Image assets
```

## Technologies Used

- HTML5
- TailwindCSS (CDN)

## Usage

Simply open `index.html` in a web browser to view the portfolio.
